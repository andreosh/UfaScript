# UfaScript
* UfaScript.osb - библиотека для OneScript, позволяет организовать консольный интерфейс, возможность создания полноценных объектов (ООП), передача функций в качестве параметров.
* Kubik.os - игра "Кубик-Рубик", демонстрирующая возможности консольного интерфейса (запускать под Windows Kubik.bat).
* Test.os  - Тест-файл, для демонстрации методов работы с объектами.  (запускать под Windows 1.bat)

### Установка

1. Зависимостей не имеет.

2. Библиотека UfaScript.osb должна располагаться с исполняемыми файлами os в одном каталоге.

3. Демонстрационные и обучающие видео см.на моем канале в youtube.com  Андрей Ош
