# UfaScript
* UfaScript.osb - библиотека для OneScript.
* Kubik.os - игра "Кубик-Рубик", демонстрирующая возможности консольного интерфейса (запускать под Windows Kubik.bat).
* button.os  - Шаблон с комментариями для создания кнопочного консольного интерфейса для проектов.  (запускать под Windows button.bat)
* AjaxGET.os - Пример организации GET-запроса с использованием функций библиотеки UfaScript.osb
* AjaxPOST.os - Пример организации POST-запроса с использованием функций библиотеки UfaScript.osb
* Gulp.os - Пример программы, которая следит за изменением файла в фоновом режиме и при изменении автоматически запускает обработку. Аналог программы Gulp
* AjaxChat.os - Создание простого чата с куками и обменом сервера с клиентом через Ajax POST-соединение
* Help.os - Документатор по функциям библиотеки генерирующий файл HTML со списком функций и описаний к ним. 
* OneScriptPHP.os - Пример создания вебинтерфейса для OneScript. 
* OneScript.zip - Шаблон для Joomla с OneScript-генератором файла index.php. Содержит пример Joomla.os. OneScript-генератор можно использовать, как конструктор CSS-кода. 
* СтатьиПоOneScript.txt - Описание объектной модели UfaScript


### Установка

1. Зависимостей не имеет.

2. Библиотека UfaScript.osb должна располагаться с исполняемыми файлами os в одном каталоге.

3. Демонстрационные и обучающие видео см.на моем канале в youtube.com  Андрей Ош
